### Часть 1: Основные теги HTML (краткий справочник)

HTML-теги можно условно разделить на несколько групп:

**1. Структура документа (скелет):**
*   `<!DOCTYPE html>` — объявление типа документа (для HTML5).
*   `<html>` — корневой элемент всей страницы.
*   `<head>` — "голова" документа, содержит служебную информацию (не отображается).
    *   `<title>` — заголовок вкладки браузера.
    *   `<meta>` — метаданные (кодировка, описание для поисковиков).
    *   `<link>` — подключение внешних ресурсов (чаще всего CSS).
*   `<body>` — "тело" документа. Все, что видит пользователь, находится здесь.

**2. Семантическая структура содержимого (важно для SEO и доступности):**
*   `<header>` — шапка сайта или раздела.
*   `<nav>` — навигационное меню.
*   `<main>` — основное, уникальное содержимое страницы.
*   `<section>` — логический раздел документа.
*   `<article>` — независимая, самодостаточная часть (пост в блоге, статья).
*   `<aside>` — боковая панель, дополнительный контент.
*   `<footer>` — подвал сайта или раздела.

**3. Основные теги для контента:**
*   `<h1>`, `<h2>` ... `<h6>` — заголовки (от самого важного к менее важному).
*   `<p>` — абзац текста.
*   `<a href="...">` — ссылка.
*   `<img src="..." alt="...">` — изображение (`alt` — обязательный атрибут для доступности!).
*   `<ul>`, `<ol>`, `<li>` — маркированный и нумерованный списки с элементами.
*   `<div>` — универсальный блочный контейнер (не семантический).
*   `<span>` — универсальный строчный контейнер для мелких элементов.

**4. Теги для форм (собирают данные от пользователя):**
*   `<form>` — контейнер для формы.
*   `<input type="...">` — поле ввода (текст, email, пароль, кнопка и т.д.).
*   `<textarea>` — многострочное текстовое поле.
*   `<label>` — подпись к полю формы.
*   `<button>` — кнопка.
*   `<select>`, `<option>` — выпадающий список.

### Часть 2: Идея для красивой страницы (HTML + CSS)

**Что будем делать: Личная страница-визитка (Portfolio Landing Page)**

Это отличный проект, чтобы отработать структуру, семантику и современный CSS.

**Структура страницы (HTML):**

1.  **`<header>`**:
    *   Логотип (имя или инициалы) слева.
    *   Навигационное меню `<nav>` справа (ссылки-якоря: `#about`, `#skills`, `#projects`, `#contact`).

2.  **`<main>`**:
    *   **Секция "Герой" (`<section class="hero">`)**:
        *   Большой заголовок (`<h1>`), например, "Привет, я [Имя]. Frontend-энтузиаст".
        *   Краткий подзаголовок (`<p>`).
        *   Яркая кнопка-призыв (`<a>` или `<button>`) "Посмотреть мои работы" или "Связаться со мной".
        *   Можно добавить стилизованное декоративное изображение или SVG.
    *   **Секция "Обо мне" (`<section id="about">`)**:
        *   Заголовок `<h2>`.
        *   Аватарка (`<img>`).
        *   Несколько абзацев текста.
    *   **Секция "Навыки" (`<section id="skills">`)**:
        *   Заголовок `<h2>`.
        *   Набор "карточек" (`<div class="skill-card">` или `<article>`) с иконкой и названием технологии (HTML, CSS, Git и т.д.).
    *   **Секция "Проекты" (`<section id="projects">`)**:
        *   Заголовок `<h2>`.
        *   Сетка проектов. Каждый проект — карточка (`<article class="project-card">`) со скриншотом (можно заглушка), заголовком, кратким описанием и ссылками "Код на GitHub" / "Демо".

3.  **`<footer>`**:
    *   Контактная информация или призыв к действию.
    *   Ссылки на социальные сети (иконки).
    *   Копирайт.

---

**Современные практики CSS для отработки (чтобы страница была красивой):**

1.  **CSS Custom Properties (CSS-переменные):** Задайте палитру цветов, шрифты, отступы в `:root`. Это сделает код чище и упростит изменения.
    ```css
    :root {
      --primary-color: #6366f1; /* Модный фиолетово-синий */
      --gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
      --dark-bg: #0f172a;
      --light-text: #f8fafc;
    }
    ```

2.  **Flexbox и CSS Grid:** Используйте:
    *   **Flexbox** для выравнивания элементов в хедере, навигации, в карточках навыков.
    *   **CSS Grid** для создания сетки проектов в секции "Проекты" или сложной раскладки героя.

3.  **Современные визуальные эффекты:**
    *   **Градиенты:** Для фона кнопок, заголовков (`background: var(--gradient);`) или даже для текста (`background-clip: text; -webkit-background-clip: text; color: transparent;`).
    *   **Плавные анимации при наведении (Hover-effects):**
        *   Кнопки: плавное изменение цвета фона, тени, трансформация (`transform: scale(1.05) translateY(-2px);`).
        *   Карточки проектов: легкий подъем (`transform: translateY(-10px);`), появление тени (`box-shadow`).
        *   Ссылки в навигации: подчеркивание снизу при наведении (с анимацией `width` от 0% до 100%).
    *   **Тени (`box-shadow`):** Используйте мягкие, многослойные тени для карточек, чтобы они "парили" над страницей.
    *   **Скругленные углы (`border-radius`):** Не забывайте про них (кнопки, карточки, изображения).

4.  **Типографика:**
    *   Подключите красивый шрифт с Google Fonts (например, `Inter` или `Poppins`).
    *   Поиграйте с размерами, высотой строки (`line-height`) и межбуквенным интервалом (`letter-spacing`) для заголовков.

5.  **Адаптивность (Responsive Design):** Обязательно!
    *   Используйте медиазапросы (`@media (max-width: 768px) { ... }`).
    *   Сделайте навигацию адаптивной (например, превратите меню в "гамбургер" на мобильных).
    *   Измените сетки с Grid/Flexbox на одноколоночные на маленьких экранах.

**Примерный план стилизации кнопки (чтобы было конкретнее):**
```css
.cta-button {
  display: inline-block;
  padding: 1rem 2rem;
  background: var(--gradient);
  color: white;
  border: none;
  border-radius: 50px; /* Сильное скругление */
  font-weight: bold;
  text-decoration: none;
  box-shadow: 0 4px 15px rgba(99, 102, 241, 0.3);
  transition: all 0.3s ease; /* Плавность для всех изменений */
}

.cta-button:hover {
  transform: scale(1.05) translateY(-3px);
  box-shadow: 0 10px 25px rgba(99, 102, 241, 0.5);
  /* Можно добавить animation: pulse 1s infinite; для пульсации */
}
```

**Что вы потренируете:**
*   Семантическую верстку.
*   Работу с Flexbox/Grid.
*   Подключение шрифтов и иконок.
*   Создание сложных hover-эффектов.
*   Написание адаптивного CSS.
*   Использование CSS-переменных.
*   Организацию CSS-кода.
